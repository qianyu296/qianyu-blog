# 千语博客项目需求说明书

## 1. 项目背景
千语博客是一个面向个人学习与内容沉淀的轻量博客系统。项目采用前后端分离架构，后端提供 REST API，前端提供公开浏览页面和管理员后台页面。

## 2. 项目目标
- 提供访客可访问的博客首页、分类筛选和文章详情。
- 提供管理员登录后的文章与分类管理能力。
- 使用 MySQL 存储业务数据。
- 使用 JWT 保护后台管理接口。
- 提供中文 API 说明与 OpenAPI 文档，便于导入 Apifox 调试。

## 3. 用户角色
| 角色 | 说明 |
| --- | --- |
| 访客 | 无需登录，可浏览已发布文章和分类。 |
| 管理员 | 使用固定管理员账号登录，可维护文章和分类。 |

## 4. 功能范围
### 4.1 公开博客
- 查看文章列表。
- 按分类筛选文章。
- 查看文章详情。
- 只展示状态为 `PUBLISHED` 的文章。

### 4.2 后台管理
- 管理员登录。
- 查看后台文章列表。
- 新增、编辑、删除文章。
- 新增、查看、删除分类。
- 分类下存在文章时不允许删除分类。

## 5. 非目标
- 不提供访客注册、评论、点赞、搜索、文件上传、多管理员权限体系。
- 不实现富文本编辑器，仅使用普通文本编辑文章内容。
- 不实现复杂运营统计或推荐算法。

## 6. 数据模型
### AdminUser
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | Long | 管理员 ID |
| username | String | 登录名，唯一 |
| passwordHash | String | BCrypt 密码哈希 |
| createdAt | DateTime | 创建时间 |
| updatedAt | DateTime | 更新时间 |

### Category
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | Long | 分类 ID |
| name | String | 分类名称，唯一 |
| slug | String | 分类标识，唯一 |
| description | String | 分类说明 |
| createdAt | DateTime | 创建时间 |
| updatedAt | DateTime | 更新时间 |

### Post
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | Long | 文章 ID |
| title | String | 标题 |
| summary | String | 摘要 |
| content | Text | 正文 |
| categoryId | Long | 所属分类 |
| status | DRAFT/PUBLISHED | 文章状态 |
| createdAt | DateTime | 创建时间 |
| updatedAt | DateTime | 更新时间 |

## 7. 业务流程
### 7.1 访客阅读
1. 访客进入首页。
2. 系统加载分类和已发布文章列表。
3. 访客可选择分类筛选。
4. 点击文章标题进入详情页。

### 7.2 管理员发布文章
1. 管理员进入后台登录页。
2. 登录成功后前端保存 JWT。
3. 管理员创建分类。
4. 管理员新建文章并选择发布状态。
5. 状态为 `PUBLISHED` 的文章出现在公开博客中。

## 8. 验收标准
- 后端应用可启动，并能连接 MySQL。
- 管理员可通过 `/api/admin/auth/login` 获取 JWT。
- 未携带 JWT 访问后台接口返回 401。
- 公开接口无需登录即可访问。
- 前端可浏览首页、文章详情、登录后台并进入管理页面。
- `docs/openapi.json` 可导入 Apifox。
